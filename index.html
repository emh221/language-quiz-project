A language that you might want to consider learning is:

Javacsript

JavaScript is used for all of the normal kinds of web browser activities like drop-down menus, hiding, or showing something when a button is clicked, and more! There are a variety of JavaScript frameworks that one can build websites or web applications around. Examples of them are React, Angular, or Vue.

Ruby

Ruby was created with the idea of making programming more accessible. It does not have very hard rules on how to build featrues, it allows more flexibility to solve problems using different methods. You can also build things from scratch more quickly with Ruby.

C#

C# is a general-purpose language designed for developing apps on the Microsoft platform and requires the .NET framework on Windows to work. C# has been used to create the Unity game engine. Lots of top games are made with C# and Unity, and it is applicable in VR as well.
